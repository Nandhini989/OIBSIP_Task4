# Online Examination System (Java)

## 📌 Project Overview

The **Online Examination System** is a Java console-based application that simulates an online test platform.
Users can log in, update their profile, change their password, answer multiple-choice questions (MCQs), and complete the exam within a fixed time limit.

This project was developed as part of the **Oasis Infobyte Java Development Internship**.

---

## 🎯 Objectives

The main objectives of this project are:

* Implement user authentication
* Allow users to update their profile and password
* Conduct an MCQ-based exam
* Implement a timer for the exam
* Automatically submit the exam when time expires
* Provide logout functionality

---

## ⚙️ Features

### 1. Login System

* Users must enter a valid **Username** and **Password** to access the system.

### 2. Update Profile

* Users can update their personal details such as their name.

### 3. Change Password

* Users can update their login password securely.

### 4. MCQ-Based Examination

* The exam consists of **multiple-choice questions**.
* Users select the correct option for each question.

### 5. Timer and Auto Submit

* The exam runs with a **fixed time limit**.
* If time expires, the system **automatically submits the exam**.

### 6. Logout

* Allows the user to exit the system safely.
